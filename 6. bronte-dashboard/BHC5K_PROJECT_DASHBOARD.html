    • Barcode or QR code for check-in
    • T-shirt size and start wave (if applicable)
  • Back of Bib:
    • Emergency contact information section
    • Medical conditions/allergies space
    • Timing chip attachment area (if not pre-attached)

  • Order 6-8 weeks before race to allow for design, printing, and shipping
