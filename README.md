## ğŸƒ About the Bronte Harbour Classic 5K

The **Bronte Harbour Classic 5K** is an inaugural community running event taking place on **Father's Day, June 21, 2026** at Bronte Heritage Waterfront Park in Oakville, Ontario. This family-friendly event features:

- **5K Race** - Main event for runners of all levels
- **Kids 1K Fun Run** - Family-friendly race for children
- **Full-Day Father's Day Festival** - Post-race celebration with food vendors, live music, kids activities, and community expo

### Charitable Partners

The race supports two local charities:
- **Oakville Dads** - Primary beneficiary
- **Kerr Street Mission** - Secondary beneficiary

**Target Donation:** $10,000+ combined donations

### Event Goals

- **500+ Participants** - Target registration for the inaugural event
- **Revenue Target:** $84,750-89,750 (sponsorships, registrations, vendor fees)
- **Community Impact** - Establishing an annual Oakville tradition that brings families together on Father's Day
