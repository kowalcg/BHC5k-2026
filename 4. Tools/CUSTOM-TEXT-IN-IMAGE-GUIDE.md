## üí° BEST PRACTICES

- **Include key info**: Dates, prices, event names

### **‚ùå DON'T:**
- Don't write paragraphs (keep it SHORT!)
- Don't use complex punctuation
- Don't use more than 2 lines of text
- Don't use small/subtle text (GO BIG!)
- Don't rely 100% on text accuracy (AI text rendering varies)
- Don't forget to verify spelling in generated image

## üîß TROUBLESHOOTING

### **Problem: AI misspells the text**
**Solution:** 
- Try ChatGPT or Midjourney (most accurate)
- Simplify the text (fewer words = fewer errors)
- Regenerate multiple times, pick best result
- Use ALL CAPS (helps AI recognition)

### **Problem: Text is too small**
**Solution:**
- Prompt includes "large, thumb-stopping" size instructions
- Regenerate and specify "VERY LARGE text" in special notes field
- Use platforms with better text control (ChatGPT, Grok)

### **Problem: Text hard to read against background**
**Solution:**
- Prompt includes "high contrast" and "overlay/shadow" instructions
- Try regenerating with different category (simpler backgrounds)
- Use "Include text space" checkbox for cleaner background areas

### **Problem: Text looks awkwardly placed**
**Solution:**
- Prompt includes "naturally integrated" instructions
- Try different platforms - Sora 2 and ChatGPT excel at composition
- Keep text short (easier to integrate)

### **Problem: No text appears at all**
**Solution:**
- Verify you filled the custom text field
- Check that it's not just whitespace
- Try more explicit text (avoid symbols/punctuation)
- Test with ChatGPT first (most reliable)
