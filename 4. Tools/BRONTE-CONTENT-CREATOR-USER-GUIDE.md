**Generate:** Brand new content, new settings  
**Regenerate:** Shuffle caption order, pick up any setting changes  
**Clear:** Reset everything, start fresh

Use **Regenerate** when:
- Want same captions in different order
- Made small tweaks (changed announcement, added sponsor name)
- Testing different combinations

Use **Generate** when:
- Starting completely new content
- Different category/audience/platform
- Fresh content needed

## ‚ö†Ô∏è Troubleshooting

### Issue: Button Doesn't Generate Content

**Solution:**
- Check that you selected a Content Category (required!)
- If using Custom Category, make sure you filled in the text field
- Refresh page if button is unresponsive

### Issue: Hashtags Look Similar Across Captions

**This is CORRECT!** Each caption has:
- Same core brand tags: #BronteHarbourClassic #BronteHarbour5K (always included)
- Different rotating location/race/motivation tags (unique per caption)
- Caption #1 will have #OakvilleON #BronteHarbour
- Caption #2 will have #HaltonRegion #GTA
- Caption #3 will have #BurlingtonON #LakeOntario

This is INTENTIONAL for brand consistency while reaching different micro-communities!

### Issue: Need Different Captions

**Solution:** Click **üîÑ Regenerate** button
- Shuffles captions into new random order
- Pick up any setting changes you made
- Can regenerate unlimited times

### Issue: Image Prompt Too Long

**This is CORRECT!** Professional AI prompts are detailed.
- Long prompts = better AI understanding = higher quality images
- Copy entire prompt ‚Üí Paste into AI platform
- AI platforms handle long prompts well

### Issue: Want to Start Fresh

**Solution:** Click **üóëÔ∏è Clear All Fields & Start Fresh**
- Resets everything to defaults
- Clears all text inputs
- Perfect for starting new campaign

### Issue: Wrong Race Date in Captions

**Check:**
- Is "Include race date" checked? (Should be!)
- Date is hardcoded as "June 21, 2026 (Father's Day)"
- If you need different date for specific post, edit caption after copying

### Issue: Need More Than 10 Captions

**Solution:**
1. Click Generate (get 10 captions)
2. Click Regenerate (get same 10 in new order - pick different ones!)
3. Click Regenerate again (get shuffled order - pick more!)
4. Keep clicking Regenerate = unlimited caption variations

### Issue: No Sponsor Name Showing in Captions

**Check:**
1. Did you select "ü§ù Sponsors" category?
2. Did you fill in "Featured Sponsor" field?
3. Regenerate after filling in sponsor name

### Issue: Captions Too Casual (or Too Formal)

**Solution:**
- **Too casual?** Check "Professional/formal tone" checkbox
- **Too formal?** Uncheck "Professional/formal tone" checkbox
- Regenerate to get new tone

### Issue: Too Many Emojis

**Solution:**
- Uncheck "Include emojis" checkbox
- Regenerate
- Perfect for LinkedIn, corporate, formal posts
